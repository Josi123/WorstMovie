WORST MOVIE

Este é o README do nosso aplicativo Angular usando framework Ionic. Aqui você encontrará instruções sobre como configurar e executar o aplicativo.

## Configuração

1. **Clone o repositório:** 

git clone https://github.com/Josi123/WorstMovie

2. **Instale as dependências:**  
Navegue até o diretório do seu projeto e execute o seguinte comando para instalar todas as dependências listadas no arquivo `package.json`:

cd .\WorstMovie\
npm install


## Executando o Aplicativo

Depois de instalar as dependências, você pode executar o aplicativo Angular com o seguinte comando:

ionic serve

Este comando irá compilar o aplicativo e disponibilizá-lo em um servidor local. Por padrão, o aplicativo estará disponível em `http://localhost:8100/`.

## Outras Informações Úteis

- **Versão do Node:**  
  Aplicativo foi desenvolvido com o node versão 10.7.0, recomendo o uso dessa versão ou acima.
  Para verificar sua versão do node digite no terminal npm --version
